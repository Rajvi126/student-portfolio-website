# Student Portfolio System

The Student Portfolio System is a web-based application developed using Java, JSP, and Servlets. 
It allows students to manage and display their portfolio information such as projects and details through a simple web interface.

## Features
- Add Portfolio Project
- View All Projects
- Update Project Details
- Delete Project
- Display Student Portfolio Information

## Technologies Used
- Java
- JSP (Java Server Pages)
- Servlets
- HTML & CSS
- Apache Tomcat
- Eclipse IDE

## Project Structure
StudentApp
│
├── src/main/java
│   ├── controller
│   ├── dao
│   ├── model
│   └── util
│
└── src/main/webapp
    ├── META-INF
    ├── WEB-INF
    ├── index.jsp
    ├── addProject.jsp
    ├── edit.jsp
    └── viewProjects.jsp

## How to Run the Project
1. Import the project into Eclipse IDE.
2. Configure Apache Tomcat Server.
3. Deploy the project on the server.
4. Run the application in a web browser.
